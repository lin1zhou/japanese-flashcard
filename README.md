## 特点
1. 纯前端单文件，**无任何第三方依赖、无需安装软件**
2. 体积仅约 100kb，轻便小巧
3. **完美支持离线使用**：下载 sakura_gramma.html 本地保存，断网状态下浏览器即可正常打开复习
4. 电脑、手机浏览器全适配，翻卡刷题
## 使用方法
### 方式：离线本地使用（推荐）
1. 下载仓库内的 sakura_gramma.html 文件
2. 保存到电脑/手机，直接用浏览器打开，断网可用。
